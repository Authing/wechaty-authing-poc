# Wechaty Authing 集成 POC

## CONTRIBUTING

Fork this repo.

Set env:

```bash
export AUTHING_USER_POOL_ID="xxx"
export AUTHING_USER_POOL_SECRET="xxx"
export PADLOCAL_TOKEN="xxx"
```

Start up:

```bash
npm start
```

## 说明

POC 项目中会出现大量 Hard Coding 的代码。在后续的开发中，我们会尽量避免这种硬编码。

## 以微信群作为上游数据源

修改 `src/index.ts`：

```ts
import './upstream';
// import './downstream';
```
